üí° Installation Note

To install VEM using the repository instead of direct download:

sudo tee /etc/yum.repos.d/vem.repo <<EOF
[vem]
name=VEM Repository
baseurl=https://vim-environment-manager.github.io/packages/repo/rpm
enabled=1
gpgcheck=0
EOF

sudo dnf install vem
